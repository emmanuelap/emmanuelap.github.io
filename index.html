HTML5

El lenguaje HTML5 es uno de los lenguajes más versátiles para desarrollar aplicaciones web y el más utilizado para crear apps no nativas. Esto es debido a las grandes ventajas que ofrece tanto para desarrolladores App como para clientes que quieran crear una App Android para su negocio.

CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML (incluyendo varios languages basados en XML como SVG, MathML o XHTML).

PHP y Base de Datos

¿Qué se puede hacer con PHP? PHP está enfocado principalmente a la programación de scripts del lado del servidor, por lo que se puede hacer cualquier cosa que pueda hacer otro programa CGI, como recopilar datos de formularios, generar páginas con contenidos dinámicos, o enviar y recibir cookies.

WordPress

WordPress es un software gratuito o herramienta de gestión de contenidos, también llamado CMS (Content Management System), diseñada para crear y mantener fácilmente un Blog, un sitio Web o una Tienda Online. Esta plataforma vio la luz en el año 2003.

Contenido multimedia y dinamico

Una definición simple. El contenido dinámico es una página web o un elemento del correo electrónico que cambia y adapta el contenido en función de las páginas que se visitan, los productos que agregan al carrito y el tiempo que pasan en el sitio.
